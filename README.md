# CodeAlpha_Network-Intrusion-Detection-System
A network-based IDS using Suricata/Snort with automated alert rules, real-time traffic monitoring, intrusion response mechanisms, and an attack visualization dashboard.
# Features:
- 🔎 **Real-Time Traffic Monitoring** — Continuous packet inspection on 
  live network interfaces
- 📋 **Custom Rule Engine** — Configured Snort/Suricata rules to detect 
  port scans, DoS attacks, malware signatures, SQL injection, and more
- 🚨 **Automated Alerting** — Instant alerts via logs, email, or webhook 
  on threat detection
- 🔒 **Response Mechanisms** — Automated IP blocking, session termination, 
  and firewall rule injection upon confirmed intrusion
- 📊 **Attack Visualization Dashboard** — Grafana/ELK-based dashboards 
  showing attack timelines, threat categories, geo-IP maps, and traffic 
  anomalies
- 🗂️ **Structured Log Management** — Centralized logging with JSON output 
  for easy SIEM integration
